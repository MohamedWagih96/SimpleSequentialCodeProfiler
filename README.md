# Simple Sequential Code Profiler

A static sequential code analyzer


### Prerequisites

<ul>
  <li>tabulate</li>
  <li>numpy</li>
  <li>networkx</li>
  <li>matplotlib</li>
</ul>

Download the notebook and run it that Simple :D
